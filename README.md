Book Finder :

Book Finder is a straightforward web application designed to help users discover their next great read and receive book recommendations. Whether you’re an avid reader or just looking for something new, Book Finder has you covered.

Features :

Search Books: Enter keywords, titles, or author names to find relevant books.
View Details: Get detailed information about each book, including title, author, description, page count, and average rating.
Responsive Design: Book Finder works seamlessly on desktops, tablets, and mobile devices.

Built With :

Google Books API: We leverage the Google Books API to fetch book data based on user queries.
ReactJS: The frontend is built using React, allowing for dynamic and interactive components.
Tailwind CSS: We’ve used Tailwind CSS for styling, ensuring a clean and modern look.

Installation :

Clone this repository to your local machine.
Navigate to the project directory.
Run npm install to install the necessary dependencies.
Start the development server with npm start.

Usage :

Visit the live Book Finder website.
Enter keywords or book titles in the search bar.
Explore the search results and click on any book card to view details.
